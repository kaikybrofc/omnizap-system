# Security Policy

## Supported Versions

Security fixes are applied to the `main` branch and the latest published release.

## Reporting a Vulnerability

Please do not open public issues for security reports.

Report vulnerabilities by email:

- `security@omnizap.shop`

If needed, include:

- affected component and version
- reproduction steps
- impact and suggested mitigation

We will acknowledge receipt as soon as possible and coordinate a responsible disclosure timeline.
