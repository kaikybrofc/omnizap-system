OmniZap API Docs

API pública para o catálogo de sticker packs e assets.

Base URL

https://omnizap.shop/api/sticker-packs

Listar packs

GET /api/sticker-packs?q=&visibility=public|unlisted|all&limit=&offset=

Listar stickers sem pack

GET /api/sticker-packs/orphan-stickers?q=&limit=&offset=

Detalhes de pack

GET /api/sticker-packs/:packKey

Imagem de sticker

GET /api/sticker-packs/:packKey/stickers/:stickerId.webp
GET /data/stickers/:owner/:file.webp

Resumo de métricas do sistema

GET /api/sticker-packs/system-summary

Exemplo cURL

curl -sS "https://omnizap.shop/api/sticker-packs?visibility=public&limit=5"
curl -sS "https://omnizap.shop/api/sticker-packs/orphan-stickers?limit=20&offset=0"
curl -sS "https://omnizap.shop/api/sticker-packs/system-summary"
