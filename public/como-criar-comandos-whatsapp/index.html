Perguntas frequentes

Preciso programar para ter comandos no grupo?

Nao para uso comum. O bot pronto entrega comandos basicos sem necessidade de codigo.

Comando serve apenas para grupos grandes?

Nao. Mesmo grupos pequenos ganham produtividade com respostas padronizadas.

Posso integrar comandos com API depois?

Sim. Se precisar evoluir para integracao, use a Area de Desenvolvedor em /api-docs/.
