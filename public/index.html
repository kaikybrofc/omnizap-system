Sobre o Sistema

Visão Geral

O OmniZap é uma plataforma open source para operação de bots e automações WhatsApp com base técnica orientada a produção.

Para Quem É

Projetado para devs, equipes de produto, empresas e operadores SaaS que precisam de automação confiável em alto volume.

Diferencial Técnico

Camadas de observabilidade, fila de escrita, reconciliação LID/JID e serviços modulares para reduzir risco operacional.

Estrutura Modular

Domínios de comando isolados por módulos, facilitando manutenção, evolução incremental e integração com novos recursos.

Infraestrutura e Arquitetura

Camada de Execução

Node.js com Baileys para conexão com WhatsApp, controle de sessão, recuperação e reconexão resiliente.

Persistência

MySQL com schema evolutivo, fila de escrita e proteção de consistência para chats, mensagens e metadados.

Módulos de Domínio

Administração de grupos, mídia, IA, stickers, ranking e utilitários operam como blocos independentes.

Operação em Produção

Logs estruturados, limites de retry e fluxo de monitoramento para suportar crescimento com previsibilidade.

API e Integrações

Endpoints Públicos

APIs HTTP para catálogo, ranking, status do sistema e dados de descoberta consumíveis por frontends e integrações.

Integração com Serviços

Fluxos preparados para conectar monitoramento, pipelines de conteúdo, serviços de IA e rotinas externas.

Governança de Acesso

Rotas técnicas e documentação organizada para facilitar controle de consumo e expansão por ambiente.

Pronto para SaaS

Base adequada para produtos multi-módulo com evolução incremental sem reescrever o núcleo do sistema.

Stack Tecnológica

Node.js + Baileys

Runtime principal de mensageria, orquestração de eventos e automações operacionais de WhatsApp.

MySQL + Migrations

Persistência transacional com versionamento de schema para manter evolução segura do sistema.

Observabilidade

Métricas e endpoints operacionais compatíveis com Prometheus, Grafana e monitoramento de latência.

Frontend Web

Painéis web para catálogo, documentação e administração, consumindo os mesmos serviços de API.

Módulo Secundário
