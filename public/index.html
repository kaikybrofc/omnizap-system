OmniZap System
API GitHub
Open Source SaaS-Ready WhatsApp Automation

Automação WhatsApp para bots, operações e empresas com stack pronta para produção.

O OmniZap System é uma plataforma open source para centralizar atendimento automatizado, execução de comandos, rotinas operacionais e integração com APIs em um único projeto.

Abrir Plataforma Explorar API

Visão do Projeto

Base Única de Operação

Consolida automação, comandos, mídia e integrações em uma stack única para reduzir complexidade operacional.

Arquitetura Modular

Projeto estruturado em módulos para facilitar manutenção, extensão de funcionalidades e evolução contínua do bot.

Pronto para Produção

Direcionado para uso real: foco em estabilidade, controle de fluxo, visibilidade e facilidade de deploy.

Open Source

Código aberto para customização completa, integração com sistemas próprios e colaboração da comunidade.

Recursos Principais

Automação de Comandos

Módulos para grupos, administração, rotinas de mídia e comandos para fluxos do dia a dia.

Integrações e API

Endpoints para consulta de dados, integração com serviços externos e consumo de recursos públicos do sistema.

Gestão de Conteúdo

Organização de assets e packs para manter o conteúdo do bot estruturado e fácil de reutilizar.

Observabilidade

Métricas e dados operacionais para monitorar saúde da aplicação, desempenho e capacidade.

API-First

GET /api/sticker-packs?visibility=public&limit=24&offset=0

Catálogo de Stickers

Packs Públicos

Listagem com busca e ordenação para navegar por packs compartilhados e abrir detalhes de cada coleção.

Compartilhamento Direto

Cada pack possui link dedicado, facilitando divulgação e uso no WhatsApp com comando pré-preenchido.

Visualização Web

Página de catálogo otimizada para desktop e mobile, com carregamento progressivo e navegação rápida.

Modo Sem Pack

Área específica para figurinhas sem pack, útil para curadoria e organização do conteúdo recebido.
